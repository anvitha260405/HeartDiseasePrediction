# Heart Disease Prediction using Machine Learning

## Overview

This project predicts whether a person is likely to have heart disease based on medical parameters using Machine Learning.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Logistic Regression
- Jupyter Notebook

## Dataset

The dataset contains 1025 patient records and includes features such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Maximum Heart Rate
- Exercise Induced Angina
- ST Depression
- Thalassemia
- Heart Disease Target

## Machine Learning Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Train-Test Split
6. Logistic Regression Model Training
7. Model Evaluation
8. Heart Disease Prediction

## Results

- Successfully trained a Logistic Regression classifier.
- Predicted heart disease risk using patient health parameters.
- Implemented an interactive prediction function for user input.
